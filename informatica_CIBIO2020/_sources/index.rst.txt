Note Introduttive
-----------------

* Assumeremo di lavorare *esclusivamente* con file di testo, indipendentemente
  dall'estensione dei file.
* La shell e Python non capiscono le lettere accentate, non usatele.
* Il comportamento di alcuni comandi puo' variare tra varianti diverse di Unix,
  ad esempio GNU/Linux e MacOS X.
